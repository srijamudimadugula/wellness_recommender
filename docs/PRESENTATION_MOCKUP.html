The Problem: Why Current Search Fails Wellness

  • Decision Fatigue (Cognitive Load): In moments of high stress, users have minimal mental energy. Searching through thousands of irrelevant results actually increases distress.
  • The Semantic Gap: Searching for emotions like *"I'm angry"* returns literal matches (news/rants) instead of the **relaxing relief** (Yoga/Meditation) the user actually needs.
  • Keyword vs. Emotional Need: Traditional AI focuses on what you say but ignores how you feel, creating a massive barrier to immediate emotional recovery.

Architecture: Hybrid Bayesian AI

  • Intake: NLP-driven emotion detection.
  • Filtering: Domain-restricted Yoga/Meditation queries.
  • Retrieval: Real-time YouTube API (restricted to wellness).
  • Personalization: LinUCB for ranking preferred yoga styles.

The Bridge:
Emotion Text → Wellness Domain Filter → Curated Video Ranking

NLP: Extracting the Need

Stressor Extraction
